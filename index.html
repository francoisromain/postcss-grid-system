postcss-structure

Github | Twitter

A PostCSS plugin to create CSS grids based on a fixed column width.

The rule:
The fewer differences in the size of the illustrations,
the quieter the impression created by the design.

Josef Muller Brockmann in The designer and the grid.

Included

  • Blocs
  • Bloc fractions
  • Blobs
  • Show
  • Hide
  • Align right
  • Columns
