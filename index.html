A PostCSS plugin to create CSS grids based on a fixed block width.

Included

  • Blocs
  • Bloc fractions
  • Blobs
  • Columns
  • Utilities

Blocs

bloc-(screen-size)-(width)-(offset).

Blocs width does not change depending on the sreen-size.

  • screen-size: from which the bloc width is active. Under this size, the bloc takes full width.
  • width: width of the bloc.
  • offset (optional): if screen-size is smaller than width + offset, then width shrinks first.

Blobs

blob-(screen-size)-(ratio)-(divider)

Unlike blocs, blobs width will vary depending on the screen-size.

  • screen-size: number of blocs that fit in the screen
  • ratio: fraction of the divider in one row
  • divider: aribitrary number to divide one row

Columns

columns-(screen-size)-(width)-(offset)

  • screen-size: from which the bloc takes its width. Under this size, the bloc takes full width.
  • width: number of columns.
  • offset (optional): if screen-size is smaller than width + offset, then width shrinks first.
